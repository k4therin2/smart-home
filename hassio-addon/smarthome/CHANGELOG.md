# Changelog

## 1.0.0 (2025-12-29)

### Added
- Initial Home Assistant add-on release
- Full SmartHome Assistant functionality
- Ingress support for embedded UI
- Configuration UI with translations
- Health check and watchdog support
- Multi-architecture support (amd64, aarch64, armv7)

### Features
- Natural language control of smart home devices
- Voice control integration via Home Assistant Assist
- Cost tracking and alerts for API usage
- Spotify music control integration
- Privacy-focused design with local data storage

### Configuration Options
- OpenAI API key and model selection
- Daily cost target and alert thresholds
- Slack webhook for notifications
- Spotify integration credentials
- Configurable log level
